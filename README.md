# Calculadora Simple con Clases en Python (POO)

Ejemplo educativo y 100% funcional de **Programación Orientada a Objetos** que crea una clase `Calculator` para realizar las 4 operaciones básicas: **suma, resta, multiplicación y división**.

### Qué hace este código exactamente
- Define una clase `Calculator` que recibe dos números al crearse
- Guarda los números como atributos internos (`_a` y `_b`)
- Tiene 4 métodos:
  - `.suma()` → devuelve a + b
  - `.resta()` → devuelve a − b
  - `.multiplicacion()` → devuelve a × b
  - `.divison()` → devuelve a ÷ b
- Crea un objeto con los valores 5 y 8
- Muestra los resultados de todas las operaciones
